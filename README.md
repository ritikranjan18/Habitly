# 🚀 Habit Tracker App

A modern Habit Tracker built using MERN Stack with streak tracking and beautiful UI.

---

## ✨ Features

* 🔐 User Authentication
* 🔥 Daily Streak Tracking
* 📊 Progress Dashboard
* 🎯 Category-based Habits
* 🌙 Dark Mode

---

## 🛠 Tech Stack

* Frontend: React.js
* Backend: Node.js + Express
* Database: MongoDB
